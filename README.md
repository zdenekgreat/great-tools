# Great Tools

Sada online nástrojů pro každodenní úkoly – úprava obrázků, videí, PDF, textu, seznamů a dat, vše přímo z vašeho prohlížeče.

Všechny soubory jsou zpracovávány výhradně na straně klienta: nic nikdy neopustí vaše zařízení.

![img.png](docs-images/img.png)

## Funkce

### **Obrázkové / Video / Audio nástroje**
- Změna velikosti obrázků
- Převod obrázků
- Editor obrázků
- Oříznutí videa
- Extrakce zvuku
- A další...

### **PDF nástroje**
- Rozdělení PDF
- Sloučení PDF
- PDF Editor
- A další...

### **Textové / Seznamové nástroje**
- Převod velikosti písmen
- Zamíchání seznamu
- Formátování textu
- A další...

### **Datum a čas**
- Kalkulačky data
- Převod časových zón
- A další...

### **Matematické nástroje**
- Generování čísel
- Ohmův zákon
- A další...

### **Datové nástroje**
- JSON nástroje
- CSV nástroje
- XML nástroje
- A další...

## Instalace a spuštění

```bash
git clone https://github.com/zdenekgreat/omni-tools.git
cd omni-tools
pnpm install
pnpm dev
```
